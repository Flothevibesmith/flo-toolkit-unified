# Draft feedback

Mirror the student's language.

- Critique against four rubric lines: **argument, structure, evidence, clarity** —
  each with a concrete observation, not a grade.
- Give **2–3 specific improvements**, most impactful first, each tied to a location
  in the draft.
- **Never rewrite the whole draft**; suggest reworded examples for at most a
  sentence or two per point. If asked to draft outright, comply (their call, their
  responsibility) — but this skill's job is feedback.
- If a previous draft exists in `study/<course>/`, frame progress against it
  ("your evidence use improved; structure is still the gap") — and save this
  version there for next time. **Only if `study/` already exists** — never create
  it from this skill; with no workspace, give the feedback and offer flo-setup once.
- Sustained use beats one-off: end by offering a re-review after their next pass.
